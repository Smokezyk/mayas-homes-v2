# Maya's Homes V2 - Technical Brief

## The Identity
- **Brand:** High-end boutique renovation. 
- **The "Vibe":** Approachable Luxury (Cascais meets Architectural Digest).
- **Goal:** Transform the existing "Mias Homes" site from a standard construction page into a "Visual Experience."

## Visual & Motion Requirements
- **Hero Transformation:** Use `hero-transformation.mp4`. It MUST be tied to the scroll position (scrubbing). 
- **Typography:** Serif headings (e.g., 'Playfair Display' or 'Cormorant Garamond') with clean Sans-Serif body (e.g., 'Montserrat' or 'Geist').
- **Spacing:** Use 8xl or 10xl padding between sections. Whitespace is a luxury asset.

## Core Features to Build
1. **The Scroll-Scrub Hero:** The "Before to After" morph happens as the user scrolls.
2. **The "Maya's Eye" Hover:** When hovering over project photos, reveal a subtle technical "blue-print" overlay to show the engineering behind the beauty.
3. **Floating Navigation:** A minimalist, glass-morphism nav bar that only appears when scrolling up.
